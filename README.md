# 📊 Loan Risk Analytics Project

## 🔹 Overview
This project analyzes loan application data to identify high-risk customers and predict loan rejection.

## 🔹 Problem Statement
Financial institutions face challenges in identifying risky customers. This project builds a predictive model to improve decision-making.

## 🔹 Features
- Data Cleaning & Preprocessing
- Feature Engineering (Loan Buckets, Vintage)
- Exploratory Data Analysis
- Logistic Regression Model
- Risk Segmentation

## 🔹 Tech Stack
- Python (Pandas, NumPy)
- Scikit-learn
- Matplotlib / Seaborn

## 🔹 Results
- Achieved XX% accuracy
- Identified high-risk customers with:
  - High loan amount
  - Multiple MFIs
